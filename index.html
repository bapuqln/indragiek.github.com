PROJECT
Sonora
DESCRIPTION
Sonora is an audio player that was built on one simple premise: a more immersive and effortless way to enjoy your music. Features like a central music queue, support for a multitude of audio formats, and blistering speed make Sonora the ideal companion for everyday music listening.

The recently released Sonora 2 beta introduces a faster, more fluid UI based on Core Animation and takes full advantage of Retina displays and OS X Mountain Lion.
PROJECT
Octopus
CLIENT
Quale Apps
DESCRIPTION
Octopus is a git client for the Mac that merges git's powerful functionality with a more user friendly interface to browse and interact with repositories. It focuses on ease of use versus trying to cram in as many git features as possible.

I was responsible for developing the first version of the project in its entirety.
PROJECT
Cheddar for Mac
CLIENT
Nothing Magical
DESCRIPTION
I worked alongside Sam Soffes to bring Cheddar, a task management app with instantaneous cloud sync, to the Mac in a fully native form. Cheddar for Mac features a simple, unobtrusive interface and seamless integration with its iOS and web counterparts.

I was primarily responsible for developing the custom user interface elements.
PROJECT
Snap
DESCRIPTION
Managing your shortcuts has never been easier. Snap provides a simple interface for assigning keyboard shortcuts to apps, and even automatically assigns shortcuts to all the apps in your Dock, automatically updating the shortcuts when apps are added, removed, or rearranged.
